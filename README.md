

📘 Cadastro de Aluno
Projeto simples em Java utilizando Maven e JPA (Java Persistence API) para simular um sistema de cadastro de alunos.
O sistema permite realizar operações básicas de CRUD (Create, Read, Update, Delete) em uma tabela de alunos.

🚀 Tecnologias utilizadas
Java 17+

Maven (gerenciamento de dependências)

JPA/Hibernate (persistência de dados)

MySQL (banco de dados relacional)
